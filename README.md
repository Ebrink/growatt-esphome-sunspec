# growatt-esphome-sunspec
This Sunspec implementation is based on the repository of https://github.com/mahoekst/SunspecModbusServer but has a few key differences:
- Basic ESPhome configuration is loosely based on https://github.com/pvprodk/GrowattESPHome
- It runs on the Growatt ShineWifi-X module with ESP07s chip
- Focuses on single phase Growatt inverters (I run two 3600 TL-X inverters)
